Kali Linux Web渗透之HTTrack的使用

这是我在博客发表关于渗透测试的第一篇文章，不多说什么，况且没有什么好多说。这篇文章就教大家Web渗透测试的第一步——克隆网站。

搭建攻击端环境

从标题也可以看出，我们用到的环境系统是Kali Linux，可能有些用户还不知道Kali Linux是个什么东西，可以上百度了解一下，方便你可以看懂后面的内容。了解Kali Linux：点击这里。这里再说明一点，为什么不使用Windows进行测试，因为Kali Linux拥有特别多的工具可以使用，况且以Linux本身的特征，插入黑客插件了等特别方便，它开源系统。相比Windows的不开源，使用Kali更方便。（注：Kali Linux是专门用来进行渗透测试的系统）搭建Kali系统，可以从官网下载镜像：www.kali.org。接下来具体安装教程我这里给挂个链接，就不在多阐述，因为这不是本篇的重点。安装教程网址：https://blog.csdn.net/Withadream/article/details/126821400

安装HTTrack

这里先简单介绍一下HTTrack：HTTrack是一个免费并易于使用的线下浏览器工具，全称是HTTrack Website Copier for Windows，它能够让你从互联网上下载指定的网站进行线下浏览(离线浏览)，也可以用来收集信息(甚至有网站使用隐藏的密码文件)，一些仿真度极高的伪网站（为了骗取用户密码），也是使用类似工具做的。简单来说这个工具功能就是渗透测试中的第一步：收集信息。HTTrack是Kali中的一个内置工具，可能有些版本没有这个工具，那么我们就需要安装这个工具。打开Kali的终端，输入命令 apt-get install httrack 等待安装。安装好了，终端就恢复到了原来的状态。

使用HTTrack克隆网站

打开命令行终端，输入命令 mkdir websites 创建一个用于存放克隆网站的目录（注：不要切换目录！）这个目录的名字起什么都行。之后再终端输入命令 httrack 启动HTTrack，然后它让你输入项目的名称，你可以随便起一个，这里我输入 test  然后它让你输入存放克隆网站的路径，这里我们输入上文创建的目录的路径 /root/websites 之后它让你输入你要克隆的网站的url地址，这里我输入 https://www.baidu.com （注：输入url时必须要在网址前输入“https://”或“http://”，否则会出现错误！）之后会弹出四个选项，这里直接输入 2 这个为镜像向导模式，选择这个很方便，其他选项不在多阐述。之后它又会问你是否使用代理，这里我们如果有代理的话输入，没有就什么都不键。这个过后，它会问你导入什么类型的文件，这里我们输入 * 表示所有文件。之后那个也是什么都不输入，然后输入y开始进行攻击。等一段时间后，我们在终端输入 cd /root/web/test 我是这样的目录，你输入你之前创建的那个保存克隆网站的目录，然后输入 ls 这样就列出了所有的克隆文件，你就去可以查看它。至此，本篇文章结束。
